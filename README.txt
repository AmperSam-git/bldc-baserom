Welcome to BLDC2!
-----------------


Getting Started
---------------
Apply "BLDC2.bps" to clean (headered) copy of Super Mario World and save it in the main baserom folder as BLDC2.smc, this is important for all the metadata files to work appropriately.


Documentation
-------------
The "Docs" folder will have all the information about the things included in the baserom.


Using AddMusicK
---------------
In order for the retry system's death SFX to work the included files for AddMusicK (in "AMK Files" folder) must be included in your project.
